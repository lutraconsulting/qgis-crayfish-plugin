The Crayfish plugin aspires to be a time explorer for various grid and vector datasets within QGIS.

Crayfish supports a number of hydraulic modelling software packages which output results to XMDF, SMS DAT, GRIB, HEC2D, netCDF, Serafin and SWW.

Crayfish loads and renders results directly rather than converting them to GIS formats before viewing. This allows users to flick quickly through the various output steps in the result files and to create animations.

Check out the Crayfish project page on our website for more information.
